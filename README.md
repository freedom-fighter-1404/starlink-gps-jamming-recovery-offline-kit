## English

Offline toolkit for cases where **GPS/GNSS jamming** breaks Starlink connectivity.
It talks only to the dish on your local network (`192.168.100.1:9200`). No internet needed.

### Files (4) — in Releases
- Windows: `starlink-gps-tool_windows-x86_64.exe`
- macOS: `starlink-gps-tool_macos-universal`
- Linux: `starlink-gps-tool_linux-x86_64`
- Android: `starlink-gps-recovery_android.apk`

### Quick start (Windows / macOS / Linux)
1) Connect to **Starlink Wi‑Fi**.
2) Run the file → it opens a local UI in your browser.
3) Tap **Disable GPS**. Optional: enable **Auto Mode**.

### Quick start (Android)
1) Install the APK (sideload).
2) Connect to **Starlink Wi‑Fi**.
3) Open the app → tap **Disable GPS**. Optional: **Auto Mode**.

### Verify (recommended)
- Compare SHA‑256 with `CHECKSUMS.sha256` (in the release).
- Optional: verify `CHECKSUMS.sha256` signature using `release_keys/` (in this repo).
