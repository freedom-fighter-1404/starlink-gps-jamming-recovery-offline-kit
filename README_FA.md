# ابزار بازیابی GPS استارلینک (آفلاین)

این مخزن شامل سورس‌کد و اسکریپت‌های ساختِ ابزار آفلاین برای بازیابی اتصال استارلینک در شرایط پارازیت GPS/GNSS است.

اگر نسخهٔ آماده (یک فایل برای هر سیستم‌عامل + یک APK اندروید) را می‌خواهید:
- `public_release_repo/`

ساخت بستهٔ عمومی:
- `./scripts/build_public_release.sh`

لاگ‌های داخلی جلسه (برای انتشار نیست):
- `.session_logs/`
